## Overview
Modern e-commerce platform built with React.js frontend and Node.js backend.

## Features
- User Authentication (JWT)
- Product Catalog
- Shopping Cart
- Responsive Design
- MongoDB Integration

## Technology Stack
- **Frontend**: React.js 19.1.1, Vite, React Router, Axios
- **Backend**: Node.js, Express.js, MongoDB, JWT, bcryptjs

## Installation

### Frontend
```bash
cd shopease-frontend
npm install
npm run dev
```

### Backend
```bash
cd shopease-backend
npm install
npm start
```

## Environment Variables
```
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
PORT=3001
```
